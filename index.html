Tentang PasarLoak

Apa itu PasarLoak?

PasarLoak adalah platform marketplace eksklusif yang didedikasikan untuk pencinta barang thrift (bekas berkualitas) dan kolektor barang antik (vintage). Kami hadir untuk memberikan nilai baru pada barang-barang yang memiliki sejarah, menggabungkan nuansa retro dengan kenyamanan teknologi modern.

Tujuan Pengembangan

Website ini dikembangkan sebagai Proyek Tugas Akhir Mata Kuliah Pengembangan Web. Tujuan utama dari pembuatan platform ini adalah:

  • Implementasi desain responsif menggunakan HTML5 dan CSS dan JavaScript.
  • Penerapan logika program Single Page Application (SPA) dengan JavaScript.
  • Manajemen data interaktif menggunakan sistem LocalStorage.
  • Studi kasus integrasi User Experience (UX) pada platform e-commerce.
